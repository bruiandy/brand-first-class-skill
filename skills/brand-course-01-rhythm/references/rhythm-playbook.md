# 节奏感判断说明

## 主原则

初创品牌最容易出问题的，不是动作不够，而是顺序错了。

这节课的正确顺序是：

1. 认知：为什么做
2. 战略：做什么
3. 运营：怎么做
4. 业务：谁来做，怎么推进，怎么落到组织和动作上

如果用户一上来就在聊“谁来做”和“怎么做”，要把问题拉回前两步。

## 为什么顺序不能乱

- 没想清楚为什么做，后面所有动作都容易失焦
- 没想清楚做什么，渠道、内容、团队配置都会跑偏
- 直接从运营和业务开始，只会让战术代替战略
- 团队越强、执行越快，方向错了反而损失越大

## 典型错法

一个典型场景是：工厂老板想做自有品牌，一上来先招市场总监、做抖音、找网红、谈联名，团队忙了几个月，钱花了不少，但“为什么做这个品牌、到底要做成什么”始终没想清楚。最后看上去动作很多，实际上主线是空的。

这类问题的本质，不是执行不努力，而是顺序从后往前倒了。

## 适合追问的问题

- 你为什么现在一定要做这个品牌？
- 你真正要做的，是一个产品，一个渠道模型，还是一个长期品牌资产？
- 在所有执行动作之前，哪一个判断还没有清楚？
- 你现在讨论最多的，是战略问题，还是执行问题？

## 输出模板

- 当前节奏：现在卡在哪一层
- 跳过的步骤：认知 / 战略 / 运营 / 业务
- 应该先补的判断
- 第一动作
- 暂时不要做的动作

## 什么时候不要单独用这一节

如果用户的问题已经进入“我们到底靠什么赢”或“这个经营模型能不能成立”，要继续去：

- `brand-course-02-resource-scan`
- `brand-course-03-operating-think`

## 什么时候继续往下走

当用户已经把顺序摆正，明确知道现在该先回答“为什么做”和“做什么”，下一步就该继续判断：

- 我们真正有什么牌可打
- 哪一张牌可能成为不公平优势
- 这个经营模型到底成不成立

如果问题更偏资源和长板，进入 `brand-course-02-resource-scan`。  
如果问题已经落到品类、渠道、团队和毛利现实，进入 `brand-course-03-operating-think`。
