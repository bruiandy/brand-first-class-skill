# 经营判断矩阵

## 顶层先看四个元素

这节课先看四个顶层元素：

1. 品类
2. 渠道
3. 团队
4. 毛利

先用这四个词把经营现实看清，再往下展开判断维度。

## 1. 品类

看什么：

- 市场集中度
- 线上渗透率
- 流量敏感度
- 信息对称性

这决定了品牌需要的是差异化、效率、信任，还是更重的场景教育。

## 2. 渠道

看什么：

- 当前主渠道是什么
- 线上和线下谁更有话语权
- 渠道是强分发，还是强内容承接
- 产品和渠道是否匹配

这决定了增长应该从哪里打穿，而不是一开始全渠道开花。

## 3. 团队

看什么：

- 当前组织更强产品、销售、内容还是运营
- 团队能力是否匹配当前主渠道
- 有没有能力承接多渠道或复杂打法

这决定了你现在能跑多复杂的模式。

## 4. 毛利

看什么：

- 当前毛利结构
- 是否支撑投流、达人、铺渠道或多团队协同
- 容错空间有多大

这决定了动作可以有多激进，还是必须先求效率。

## 输出模板

- 品类判断
- 渠道判断
- 团队判断
- 毛利判断
- 当前更可行的经营路径
- 现在不建议做什么

## 不要这样用

- 不要拿单一趋势去概括所有品牌。
- 不要默认品牌建设能自动解决渠道和毛利问题。
- 不要把一个讲得好听的故事，误当成可成立的经营模型。

## 什么时候继续往下走

当经营现实已经大致看清，团队知道这个品牌该怎么打、哪些动作现在能做、哪些动作现在不能做，下一步就该进入“再说清”的阶段，继续回答：

- 这个品牌为什么存在
- 它到底服务谁、坚持什么
- 它应该如何把自己说清楚

通常先进入 `brand-course-04-brand-ultimate-question`。  
如果底层命题已经比较清楚，也可以继续去 `brand-course-05-brand-narrative-system` 或 `brand-course-06-product-value-shaping`。
