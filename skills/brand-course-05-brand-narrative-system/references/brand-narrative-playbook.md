# 品牌叙事体系说明

## 话语体系不是写几句文案

这一节要做的，是把品牌最核心的价值命题，稳定地展开成一套可被用户反复感知的表达系统。

先后顺序通常是：

1. 定位
2. 命名
3. Slogan
4. 视觉符号
5. 关键触点

## 每一层在回答什么

### 定位

回答：你在用户心里要占据什么位置。

### 命名

回答：用户如何记住你、称呼你。

### Slogan

回答：用户最快能抓住你哪一个价值。

### 视觉符号

回答：用户看到什么就能想起你。

### 关键触点

回答：用户会在哪些关键场景里反复接收到同一套信息。

## 常见错法

- 先起名字，再倒推品牌逻辑。
- 定位一句话很宏大，但落不到包装、页面、销售和内容。
- Slogan 追求文艺或高级，却没有记忆点和行动价值。
- 视觉上很好看，但和核心价值没有关系。
- 每个触点各说各话，品牌无法形成复利。

## 适合追问的问题

- 用户第一次接触你时，最该记住的那一句话是什么？
- 你最不希望被误解成谁？
- 命名和 slogan 是否都在放大同一个价值？
- 包装、页面、内容和销售话术是否在讲同一件事？
- 哪些触点最关键，值得反复强化？

## 输出模板

- 核心价值主线
- 定位句
- 命名方向
- Slogan 方向
- 视觉符号方向
- 关键触点建议
- 目前最不一致的表达缺口

## 什么时候继续往下走

当品牌已经把“你是谁、用户为什么该记住你”说清楚，下一步就该回到产品本身，回答：

- 用户到底在什么场景下雇佣这个产品
- 他买的不是功能，而是从 A 状态到 B 状态的哪张船票
- 品牌承诺和产品体验是否一致

也就是 `brand-course-06-product-value-shaping`。
