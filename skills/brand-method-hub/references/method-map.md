# 品牌第一课方法地图

## 核心口径

对外网站可以讲 `6 个品牌经营基本功`，这是传播结构。  
Skill 系统不按网站结构走，而按课程产品结构走：

开篇 + 7 个章节。

## 课程主链

1. 开篇：经营主线和节奏感
2. 第一节：盘资源，找到核心优势
3. 第二节：建立业务经营思维
4. 第三节：品牌终极之问
5. 第四节：构建品牌叙事体系
6. 第五节：产品价值塑造
7. 第六节：产品与渠道闭环
8. 第七节：最小营销闭环

## 支撑工具

- `品牌第一课｜从这里开始`：用户不知道从哪开始时使用。
- `brand-foundation-diagnosis`：用户带着真实案例、问卷或增长焦虑时使用。
- `brand-case-lens`：用户想拆外部案例、写内容、做市场观察时使用。
- `brand-method-hub`：用户想理解整套方法、检查 Artifact 链、做路由复盘时使用。

## 路由规则

- 问“我现在该先做什么”：进入开篇节奏判断。
- 问“我到底有什么优势”：进入盘资源。
- 问“这个生意模型能不能成立”：进入经营思维。
- 问“品牌为什么存在”：进入品牌终极之问。
- 问“定位、命名、Slogan、视觉和服务怎么统一”：进入品牌叙事。
- 问“产品到底卖什么价值”：进入产品价值。
- 问“产品适合哪个渠道”：进入产品渠道闭环。
- 问“怎么跑通第一条增长路径”：进入最小营销闭环。

## 判断原则

不要越节回答。  
不要把后面几节一次讲完。  
每一节都应收束成自己的沟通纪要，再交给下一节读取。
